# ChatGPT 自動化スクリプト

このフォルダには、ChatGPTをブラウザで自動操作するためのPythonスクリプトが含まれています。
セキュリティ上の理由（ログイン情報の取り扱い）により、AIエージェントが直接ログインを行う代わりに、
お客様のローカル環境で実行できるスクリプトを作成しました。

## 前提条件

*   Python がインストールされていること
*   Chrome または Chromium ベースのブラウザ

## セットアップ方法

以下のコマンドをターミナルで実行して、必要なライブラリをインストールしてください。

```bash
pip install -r requirements.txt
playwright install
```

## 実行方法

スクリプトを実行します。

```bash
python automate_gpt.py
```

## スクリプトの動作

1.  ブラウザ（Chromium）が起動し、ChatGPTのトップページが開きます。
2.  **【手動操作】** 画面にてログインを行ってください。
3.  ログイン完了後、ターミナルに戻り `Enter` キーを押してください。
4.  スクリプトが自動的に以下の操作を行います：
    *   GPTストアへ移動
    *   「ワークコンパスAI」を検索・選択
    *   「SNS運用始めたい」と入力して送信（送信前に確認が入ります）
