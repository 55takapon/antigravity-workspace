# 実装計画書：お好み焼き店「みち」LP制作

加古川のお好み焼き店「みち」の魅力を伝える、高品質な1ページLPを制作します。「和モダン」かつ「温かみ」のある、日本のプロデザイナーが手がけたようなデザインを目指します。

## ユーザーレビューが必要な項目
- デザインのトーン＆マナー（和風・温かみ・清潔感）
- 使用するセクション構成の最終確認

## 提案される変更点

### デザインコンセプト
- **カラーパレット**: 深い茶色（鉄板・木材）、温かい生成り色（壁・和紙）、アクセントに深い赤（暖簾・情熱）。
- **タイポグラフィ**: 縦書きのアクセント、信頼感のある明朝体（見出し）と読みやすいゴシック体（本文）。
- **レイアウト**: 余白を活かした、日本のWebデザイン特有の美学を取り入れます。

### 構成セクション
1. **Hero**: フルスクリーン画像 + 縦書きキャッチコピー + クイック情報（電話・時間）。
2. **Concept**: 夫婦の想いを伝える、余白たっぷりのタイポグラフィセクション。
3. **USP**: アイコン付きの3つの強み。
4. **Menu**: 写真を大きく配置したグリッドレイアウト。
5. **Gallery**: 店内の雰囲気を伝える清潔感のある画像。
6. **Owner**: 夫婦の温かさが伝わるストーリー風セクション。
7. **Reviews**: 信頼性を高めるクチコミ。
8. **Access**: Google Maps + 丁寧な店舗情報表。
9. **CTA**: 目立つが上品なテイクアウトボタン。

## 技術スタック・設計
- **HTML5/CSS3**: セマンティックなマークアップ。
- **CSS設計**: FLOCSS または BEM をベースにした保守性の高い設計。
- **レスポンシブ**: モバイルファーストで、全デバイスで最適な表示。
- **フォント**: Google Fonts (Noto Serif JP, Noto Sans JP)。

## 検証プラン
### 自動テスト/チェック
- `npm run lint` 等の静的解析（必要に応じて）。
- デバイスエミュレーションを用いた全画面チェック。

### 手動検証
- 開発者ツールによるレスポンシブ崩れの確認。
- 画像の読み込み速度とアスペクト比の維持の確認。
- Canvasでの最終プレビュー。
