# 鳥縁（とりえん）ウェブサイト制作完了レポート

## 制作概要

炭火焼き鳥居酒屋「鳥縁（とりえん）」のトップページを、日本のプロデザイナー品質で制作しました。

### 制作方針
- **デザイン**: 和モダンで高級感のある雰囲気
- **ターゲット**: 炭火焼の地鶏が好きな方
- **目的**: お客様が予約したくなるキャッチーなファーストビュー
- **コード品質**: プロのコーダーが書くクリーンなコード

## 実装内容

### 1. CSS設計（BEM + FLOCSS）

プロフェッショナルなCSS設計手法を採用しました。

#### FLOCSS構造
```
Foundation - リセット・ベーススタイル
Layout - ヘッダー、フッター、メインレイアウト
Object
  ├─ Component - 再利用可能なコンポーネント
  ├─ Project - プロジェクト固有のスタイル
  └─ Utility - ヘルパークラス
```

#### BEM命名規則
- Block: `.c-btn`, `.p-hero`
- Element: `.c-btn__text`, `.p-hero__content`
- Modifier: `.c-btn--primary`, `.l-section--dark`

#### カラーパレット
- **メインカラー**: `#8B4513` (炭火の茶色)
- **アクセントカラー**: `#D4AF37` (金色 - プレミアム感)
- **ダークカラー**: `#2C1810` (深い焦げ茶)
- **ライトカラー**: `#F5F5DC` (ベージュ - 和紙のような温かみ)

### 2. HTMLマークアップ

セマンティックHTML5で構造化しました。

#### セクション構成
1. **ヘッダー** - 固定ナビゲーション
2. **ヒーローセクション** - キャッチーなファーストビュー
3. **コンセプトセクション** - 3つのこだわり（厳選地鶏、炭火焼き、職人の技）
4. **メニューセクション** - おすすめメニュー3品
5. **こだわりセクション** - 鳥縁の想い（産地直送、備長炭、店内雰囲気）
6. **アクセスセクション** - 店舗情報と予約CTA
7. **フッター** - 店舗情報

### 3. レスポンシブ対応

モバイルファーストで設計し、4つのブレークポイントを設定しました。

#### ブレークポイント
- **モバイル**: デフォルト（〜767px）
- **タブレット**: 768px以上
- **デスクトップ**: 1024px以上
- **ワイドデスクトップ**: 1440px以上

#### レスポンシブ機能
- フレキシブルグリッドレイアウト
- 可変画像（`max-width: 100%`）
- メディアクエリによる最適化
- タッチデバイス対応

## コード品質

### CSS設計の特徴

#### 1. 明確な責任分離
```css
/* Foundation - 基礎スタイル */
*, *::before, *::after { box-sizing: border-box; }

/* Layout - レイアウト */
.l-header { position: fixed; }

/* Component - 再利用可能 */
.c-btn { display: inline-block; }

/* Project - プロジェクト固有 */
.p-hero { position: relative; }

/* Utility - ヘルパー */
.u-mt-1 { margin-top: 1rem; }
```

#### 2. 保守性の高い命名
- 接頭辞による分類（`l-`, `c-`, `p-`, `u-`）
- BEM記法による明確な関係性
- 意味のある名前

#### 3. レスポンシブ設計
