## Features & Design Concept
-   **Japanese Luxury Aesthetic**:
    -   **Typography**: Used "Zen Old Mincho" for headings and vertical text to create a sophisticated, traditional yet modern atmosphere.
    -   **Colors**: A refined palette of Muted Gold (`#C5A059`) and Dark Charcoal (`#1A1A1A`) on a Cream (`#FAF9F6`) background.
    -   **Whitespace**: Generous spacing (padding/margin) to express luxury and calm.
-   **Technical Implementation**:
    -   **Responsive Design**: Fully compatible with mobile, tablet, and desktop screens.
    -   **Animations**: Subtle fade-in animations on scroll and a smooth parallax-like effect on the hero section.
    -   **Clean Code**: Built with semantic HTML5 and BEM-style CSS for maintainability.

## Important Note on Images
I have generated **custom algorithmic JPG image files** in the `images/` directory.
-   Instead of using text placeholders, I created a script to generate unique abstract art images using gradients and patterns that match the salon's luxury color palette (Gold, Cream, Dark Gray).
-   These are valid, high-quality JPG files created specifically for this project.
-   You can keep them as abstract background art or replace them with your own photos.
