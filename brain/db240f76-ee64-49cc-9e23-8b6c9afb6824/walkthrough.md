# Luxury Salon Website - Walkthrough

I have completed the top page for the luxury salon "LUMIÈRE".

## Features
- **Minimalist Design**: Used generous whitespace and elegant serif fonts (Mincho/Garamond).
- **Responsive**: Adjusted layout for mobile devices (horizontal text fallback).
- **Smooth Animations**: Added fade-in effects for text and buttons.
- **Fallback Hero**: Due to quota limits, I generated a custom abstract PNG background using a PowerShell script to ensure high quality without external dependencies.

## Key Files
- `index.html`: Main structure.
- `style.css`: All styles and animations.
- `assets/hero_bg.svg`: The abstract background asset.

## How to View
Open `luxury-salon-website/index.html` in your browser.

> [!NOTE]
> The hero image is currently a generated SVG pattern because the AI image generation quota was reached. In a real production scenario, this should be replaced with the high-resolution photo as originally planned.
